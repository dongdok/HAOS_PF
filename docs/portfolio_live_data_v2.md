# HAOS_Control — Infrastructure & UI Deep-Dive (포트폴리오 심화 자료)

> 데이터 기준일: 2026-03-18  
> 이 문서는 시스템의 외면(UI/UX)과 내면(인프라/거버넌스)을 심층 분석하여, 단순한 자동화를 넘어선 '플랫폼 엔지니어링' 역량을 증명합니다.

## 1. 대시보드 및 UI/UX 아키텍처 (Frontend Strategy)

### 1-0. UI 구조 정량 지표 (2026-03-19)
- Lovelace 뷰 수: **2개**
- 섹션 수: **6개**
- 카드 수: **56개**
- 뷰 타입 분포: `sections 1 / masonry 1`
- 추천 운영 카드:  
  `보완 필요 자동화`, `정상 운영 자동화`, `미구현 자동화 추천`

### 1-1. HomeKit 스타일의 구역형(Sections) 레이아웃
- **구조화된 정보 설계**: `views` 내에 `Sections` 모드를 활용하여 **집 전체-거실-주방-화장실-옷방-안방**으로 이어지는 물리적 공간 중심의 직관적 UI를 구축했습니다.
- **모던 타일(Tile) 인터페이스**: 최신 HA 디자인 표준인 Tile Card를 전면 도입하여, 기기의 상태 확인과 제어(Toggle, Features)를 동시에 수행하는 고도화된 UX를 제공합니다.

### 1-2. 데이터 시각화 및 분석 도구
- **트렌드 그래프(Trend Graph) 통합**: 거실 TV의 조도(`illuminance`)와 밝기(`brightness`) 데이터를 단순 숫자가 아닌 실시간 트렌드 그래프로 시각화하여, 환경 변화를 한눈에 파악할 수 있도록 설계했습니다.
- **조건부 기능 노출**: 커튼의 열림/닫힘 기능, 가습기 토글, HVAC 모드 선택 등 기기별 특성에 맞는 `features`를 타일 내에 정밀하게 배치했습니다.

## 2. 에너지 관리 거버넌스 (Energy Governance)

### 2-1. 계층적 전력 모니터링 체계
시스템은 개별 기기부터 집 전체 전력까지 다계층(Multi-layer)으로 모니터링하고 있습니다.
- **실시간 계측**: 안방 히터, 가습기, 전기매트, 주방 드라이기 등 핵심 전열기구의 실시간 소비전력(`kWh`)을 독립적으로 추적합니다.
- **누적 통계 엔진**: `sensor.cong_jeonryeog_eneoji_habgye`(총 합계)를 기반으로 **일간(Daily)** 및 **월간(Monthly)** 누적 데이터를 생성하는 가상 센서 레이어를 운용합니다.
  - *예시*: 월 누적 히터 전력(`1.203kWh`), 월 누적 전기매트 전력(`1.319kWh`) 등 상세 통계 추출 가능.

## 3. 상황 인지 및 사용자 컨텍스트 (Context Awareness)

### 3-1. 모바일 기기 상태 연동
사용자의 모바일 기기(iPhone, iPad)를 시스템의 중요한 센서 노드로 활용합니다.
- **기기 건전성 감시**: 배터리 잔량 및 충전 상태 뿐만 아니라, **저장소 가용 용량(`Storage`)**과 **Wi-Fi SSID** 연결 상태까지 추적하여 외출/귀가 및 기기 관리 자동화의 기초 데이터로 활용합니다.
- **네트워크 위치 인지**: 기기의 연결 방식(Wi-Fi vs Cellular)을 통해 사용자의 실내/외 위치를 보충적으로 판단합니다.

## 4. 시스템 거버넌스 및 확장성 (Governance & Scalability)

### 4-1. 자동화 실행 모드 (Execution Policy)
34개의 자동화는 작업의 성격에 따라 최적화된 실행 정책을 가집니다.
- **Single/Restart 모드**: 트리거 시 중복 실행을 막거나 새로운 트리거를 우선하는 정책을 통해 시스템 자원 낭비를 방지하고 로직의 정합성을 유지합니다.
- **HACS 기반 확장**: `HACS`를 통해 공식 기능을 넘어선 커스텀 컴포넌트와 테마를 통합 관리하며, 시스템의 확장성을 지속적으로 확보하고 있습니다.

### 4-3. 추천 엔진 운영 거버넌스 지표
- 추천 후보 누적: **50건**
- 상태 분포: `proposed 45 / rejected 3 / rolled_back 2`
- 충돌 피드백 누적: **6건** (`intended 5 / unintended 1`)
- 해석: 추천을 즉시 적용하지 않고, 제안-검토-피드백 흐름으로 안전하게 운영 중

### 4-2. 환경 변수 및 위치 추적 (Environmental IQ)
- **정밀 일조량 계산**: 시스템 위치의 고도(Elevation)와 방위각(Azimuth) 데이터를 실시간 계산(`sun.sun`)하여, 일출/일몰 기반 자동화(예: 안방 커튼 제어)의 정확도를 초단위로 유지합니다.

## 5. 핵심 기술적 발견 (Technical Discoveries)

- **UI/UX 기획력**: `lovelace_active.json` 분석 결과, 시공간적 일관성을 가진 UI 테마와 계층적 레이아웃 설계 능력이 입증됨.
- **인프라 통합력**: Zigbee, MQTT, Tuya, Cloud, Mobile App 등 이기종 플랫폼을 하나의 통제권(Governance) 아래 결합함.
- **데이터 중심 설계**: 모든 자동화의 근간에 정밀 센싱(mmWave)과 전력 데이터, 기기 상태값이 유기적으로 연결되어 있음.
