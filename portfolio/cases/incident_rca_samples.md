# 장애 RCA 샘플 (공개용)

## RCA-01 Zigbee 명령 timeout

- 증상: `Delivery failed`, `ZCL command timeout` 반복
- 영향: 특정 조명/스위치 간헐 제어 실패
- 원인: 2.4GHz 간섭 구간 + 메쉬 경로 불안정
- 조치: 채널 조정, 기기 재조인, 참조 동기화
- 재검증: 동일 제어 시나리오 반복 실행 시 실패 빈도 감소 확인

## RCA-02 재등록 후 자동화 미동작

- 증상: 기기는 조인 완료 상태이나 자동화가 대상을 찾지 못함
- 영향: 일부 자동화 미실행
- 원인: 엔티티 ID 변경 후 기존 자동화 참조 미갱신
- 조치: 캐노니컬 ID 기준으로 자동화/스크립트/대시보드 참조 동기화
- 재검증: 트레이스에서 대상 엔티티 정상 참조 확인

## RCA-03 중복 명령 전송

- 증상: 동일 대상 ON/OFF 명령 중복 전송
- 영향: 로그 노이즈 증가, 일부 지연 발생
- 원인: 상태 확인 없이 명령 전송
- 조치: "이미 목표 상태면 스킵" 조건 추가
- 재검증: 동일 시간대 명령 건수/실패 로그 감소 추이 확인
