# 실사용 자동화 예시

## 1. 밤 시간 집안 소등

- **상황**: 밤 11시에서 새벽 6시 사이, 화장실이나 주방을 거쳐 침실로 이동하는 상황.
- **동작**: 거실에 사람이 없는 것이 확인되면 거실 조명을 모두 끔. 통신 문제로 명령이 빠지지 않도록 가벼운 재시도 로직 포함.
- **의도**: 불필요하게 켜져 있는 조명을 줄이면서도, 실제 사람이 있는 구역의 조명은 유지함.

## 2. 화장실 머무는 시간에 따른 환풍기 가동

- **상황**: 화장실에 들어가 조명이 켜진 상황.
- **동작**: 조명이 켜진 지 일정 시간(약 3분 30초)이 지나면 환풍기를 자동으로 켜고, 나중에 알아서 꺼지도록 설정.
- **의도**: 짧은 용무 시에는 환풍기가 돌지 않게 하여 불필요한 작동을 피함.

## 3. 휴대폰 충전에 따른 취침 모드 오작동 방지

- **상황**: 심야 시간에 휴대폰을 충전기에 꽂을 때 취침 모드가 마음대로 켜질 수 있는 상황.
- **동작**: 취침 모드가 켜졌더라도 일정 시간 내에 침대에 사람이 있는 게 확인되지 않으면 다시 모드를 해제함.
- **의도**: 단순히 충전만 하는 것인지 진짜 잠자리에 드는 것인지 구분하여 정확도 제고.

## 4. 중복 명령 방지

- **상황**: 짧은 시간 내에 같은 자동화가 여러 번 실행될 우려가 있는 상황.
- **동작**: 이미 조명이 켜져 있거나 꺼져 있다면 불필요한 제어 명령을 보내지 않음.
- **의도**: 통신 트래픽을 줄여서 시스템의 전체적인 반응 속도를 안정적으로 유지.
