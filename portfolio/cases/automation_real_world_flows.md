# 생활 흐름 기반 자동화 예시 (공개용)

> 목적: 실제 사용자가 체감하는 자동화를 짧고 이해하기 쉽게 설명

## 1) 밤 시간 이동 후 거실 소등

- 상황
  - 밤 23:00~06:00 사이, 화장실/주방/거실을 지나 안방으로 빠르게 이동
- 동작
  - 거실 재실이 해제되고 다른 구역도 비재실이면 거실 라인/스탠드 조명을 소등
  - 소등 명령이 누락될 수 있는 구간은 짧은 재시도로 보완
- 의도
  - 이동 중 불필요한 조명 잔류를 줄이되, 재실 구역 조명은 유지

## 2) 화장실 체류 시간 기반 환풍

- 상황
  - 화장실 재실 감지로 조명 ON
  - 체류 시간이 길어지면 환풍 필요
- 동작
  - 조명 ON 상태가 일정 시간(예: 3분 30초) 지속되면 환풍기를 켬
  - 환풍기는 설정 시간 뒤 자동 OFF
- 의도
  - 짧은 이용에는 과동작을 피하고, 긴 체류에서만 환풍 자동화

## 3) 취침 오탐 방지(충전 시작 이벤트 보정)

- 상황
  - 심야 시간(23:00~06:00)에 휴대폰 충전 시작으로 취침 스위치가 켜질 수 있음
- 동작
  - 취침 스위치 ON 후 짧은 시간(예: 10초) 내 침대 재실이 없으면 취침 스위치 OFF
- 의도
  - 단순 충전 이벤트를 취침으로 오인하는 경우를 줄임

## 4) 중복 명령 방지

- 상황
  - 동일 자동화가 짧은 간격으로 여러 번 실행될 수 있음
- 동작
  - 목표 상태가 이미 같으면 ON/OFF 명령을 전송하지 않음
- 의도
  - Zigbee/Wi-Fi 트래픽과 실패 로그를 줄여 운영 품질 유지
